Tag PHP

Untuk pendeklarasian dokumen PHP, anda harus menyimpan file PHP dengan ekstensi .php, adapun kode yang akan di eksekusi oleh PHP yaitu kode yang berada di antara kode <?php dan kode ?>.

Perhatikan contoh berikut ini :

{% highlight php %} {% endhighlight %}

Selain dengan tag di atas, anda juga dapat mendeklarasikan php dengan menggunakan tag berikut ini :

{% highlight html+php %} {% endhighlight %}

PHP juga mendukung tag pembuka pendek dengan syarat pengaturan short_open_tag pada php.ini sudah enable. Contoh :

{% highlight php %} {% endhighlight %}

Direkomendasikan setiap coding PHP jangan menggunakan tag pembuka pendek!

PHP membutuhkan webserver agar bisa berjalan, jika anda belum menginstall web server di komputer anda silahkan install terlebih dahulu.

Di bawah ini adalah contoh menampilkan kata Halo Dunia! menggunakan PHP dengan berbagai tag :

{% highlight html+php %} {% endhighlight %}

Jika file yang anda buat adalah murni file PHP (tanpa HTML atau lainnya), disarankan untuk tidak menggunakan tag penutup. Contoh di bawah ini akan menampilkan Halo Dunia! PHP Keren! tanpa menggunakan tag penutup :

{% highlight html+php %}
