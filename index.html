Apa Itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa skrip yang dapat disisipkan ke dalam HTML. Dengan menggunakan PHP anda dapat membuat website yang bersifat dinamis, aplikasi berbasis website, CMS (Content Management System) ataupun lainnya.

Perbedaan antara PHP dengan bahasa skrip lainnya (misal Javascript) adalah PHP berjalan di sisi server sedangkan Javascript berjalan di sisi klien.
